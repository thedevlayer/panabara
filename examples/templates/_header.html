Help

분산된 투자 종목들을 한번에 관리하고 투자 유형별, 종목별로 자산 현황을 쉽게 파악할 수 있습니다.

사용자 등록 및 로그인
  • 사용자 등록이 필요하며, 이후 로그인 및 종목 등록, 조회 가능합니다.
등록, 수정 및 삭제
  • 투자 종목 등록
    • Details 탭에서 추가, 수정, 삭제 가능하며, 반영 이후 Home 탭 내 Refresh 클릭하여 종목명, 현재기준 금액 등을 확인할 수 있습니다.
  • 자산 현황 등록
    • Trends 탭에서 기간별 평가금액, 이전 대비 수익금액 등록, 삭제 가능합니다.
주요 기능
  • 국내주식, 해외주식, 코인으로 유형 구분
  • 투자 유형별 필터 기능
  • 전체 및 유형별 지표 확인
  • 이전 조회 시 대비 손익 증감 내역 확인
  • 각종 차트 확인
  • 종목 검색 기능
  • 종목별 상세 내용 팝업
  • 특정 항목 기준으로 우선순위 확인
  • 특별 관리 항목 체크 및 구분(예: 평생 보유 대상 종목 여부)
  • 기간별 자산 현황 차트
참고 사항
  • 국내주식, 해외주식의 경우 YahooFinance, 코인은 업비트에서 제공하는 API 를 활용합니다.
  • API 트래픽을 감안하여 최대 10 종목까지 등록, 조회 가능합니다.
  • 각종 데이터는 투자 현황의 참고용으로만 사용하시기 바랍니다.
